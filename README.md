# Kenya Scouts AI Assistant

**Jambo!**  
The **Kenya Scouts AI Assistant** is a bilingual (English & Swahili), voice-enabled, exportable AI chatbot powered by **Google Gemini 2.5-Flash** and hosted on **Apache (Kali Linux)**.

It provides **instant access to official KSA documentation**, including:
- Scout Promise & Law
- Badges & Awards
- Camps & Activities
- Leadership Training
- Community Service

---

## Features

| Feature | Description |
|--------|-----------|
| **AI-Powered** | Answers using Gemini 2.5-Flash |
| **Bilingual** | English & Swahili (toggle with flag) |
| **Voice Input** | Speak your question (mic button) |
| **Export Chat** | Download conversation as `.txt` |
| **Mobile-Friendly** | Responsive design with mobile menu |
| **Rate Limited** |  |
| **Secure** | |
